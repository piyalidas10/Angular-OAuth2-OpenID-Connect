# Angular-OAuth2-OpenID-Connect
Built secure authentication and session management UIs using Angular 19 integrating OAuth2/OpenID Connect (OIDC), MFA workflows, RBAC authorization, token refresh strategies, and enterprise-grade session security controls

1. angular-oauth2-oidc : The angular-oauth2-oidc library is widely used in real-world enterprise Angular applications for secure authentication and authorization using OAuth2 and OpenID Connect (OIDC).
2. @ng-idle/core : @ng-idle in Angular is mainly used for idle detection and session timeout management in enterprise/web applications.
3. jwt-decode : jwt-decode is commonly used in frontend applications to read and inspect JWT (JSON Web Token) contents without validating the signature. It is lightweight and mostly used in Angular, React, Vue, and plain JavaScript apps.
